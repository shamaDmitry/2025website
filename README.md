## Run locally

- `npm install` for install dependencies
- `npm run dev` then open [http://localhost:3000/](http://localhost:3000/) for `dev` mode

## Build

`npm run build`
For preview locally in "production" mode, run `npm run start`

`npm run build` command will generate the `out` folder. This folder can be deployed on any static hosting service.
